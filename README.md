# E-commerce Conversion Analytics (TFM)

## Overview
This project analyzes and predicts e-commerce conversion using data from Google Analytics 4.
It was developed as a Master’s Final Project (TFM) in Business Analytics.

The goal is to transform raw digital analytics data into actionable insights for marketing and business decision-making.

## Business Problem
- Understand user behavior across the conversion funnel
- Predict purchase probability
- Segment users based on likelihood to convert
- Support data-driven marketing strategies

## Tools & Technologies
- Python (Pandas, Scikit-learn, XGBoost)
- Google Analytics 4 data
- SQL
- Power BI (dashboard & visualization)
- Jupyter Notebook

## Project Structure
- `notebooks/` → Python analysis and machine learning models
- `assets/screenshots/` → Power BI dashboard screenshots
- `reports/` → Final project report (PDF)

## Key Outputs
- Conversion prediction model
- Customer segmentation by purchase probability
- Power BI dashboard with KPIs and insights
