# E-commerce Conversion Analytics (Master’s Thesis)

## Overview
This project focuses on analyzing and predicting e-commerce conversion using data from Google Analytics 4.  
It was developed as a Master’s Final Project (TFM) for the Master’s degree in Business Analytics.

The objective is to transform raw digital analytics data into actionable insights to support marketing and business decision-making.

## Business Problem
- Understand user behavior across the conversion funnel  
- Predict customer purchase probability  
- Segment users based on likelihood to convert  
- Support data-driven marketing and campaign optimization strategies  

## Tools & Technologies
- Python (Pandas, Scikit-learn, XGBoost)  
- Google Analytics 4  
- SQL  
- Power BI (dashboard and data visualization)  
- Jupyter Notebook  

## Project Structure
- `notebooks/` → Python analysis and machine learning models (Code)
- `PowerBI/` → Power BI dashboard (.pbix) and related assets
  - `Campaña Marketing.pbix` → Interactive Power BI dashboard
  - `Dashboard.png` → Dashboard preview screenshot
- `README.md` → Project overview and documentation  

## Key Outputs
- Conversion prediction model  
- Customer segmentation based on purchase probability  
- Interactive Power BI dashboard with key KPIs and business insights
